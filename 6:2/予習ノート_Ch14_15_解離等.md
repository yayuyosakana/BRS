# 予習ノート：「認知行動療法と精神疾患その３（認知症・パーソナリティ障害・解離症・発達障害）」Ch14/15

## キーワード
せん妄、認知症、アルツハイマー病、パーソナリティ障害（クラスターA/B/C）、解離症、肥満、摂食障害（神経性やせ症・神経性過食症）、自閉スペクトラム症、ADHD

## 1. 神経認知障害（Neurocognitive Disorders: NCDs）

### せん妄（Delirium）
| 特徴 | 内容 |
|------|------|
| 発症 | 急性（時間〜日単位） |
| 経過 | 変動性（日内変動あり、夕方〜夜に悪化） |
| 意識 | 障害あり（混乱・見当識障害） |
| 原因 | 可逆的（感染・薬剤・代謝異常・手術後） |
| 予後 | 通常可逆的 |

### アルツハイマー病（Alzheimer's Disease）
| 特徴 | 内容 |
|------|------|
| 発症 | 緩徐（年単位） |
| 経過 | 進行性・不可逆的 |
| 症状 | 記憶障害（特に近時記憶）→ 失語・失認・失行・実行機能障害 |
| 脳画像 | 海馬萎縮、脳室拡大 |
| 病理 | **アミロイド班（老人斑）**、神経原線維変化（タウタンパク） |
| 遺伝 | 染色体21（ダウン症）、早発型（Chr.1, 14）、apoE4遺伝子 |
| 経過 | 診断後8〜10年で死亡が多い |

#### アルツハイマー病の神経伝達物質
- アセチルコリン↓（最重要）、グルタミン酸異常

#### 治療
- **アセチルコリンエステラーゼ阻害薬**（ドネペジルなど）：進行を遅らせる
- **メマンチン（NMDA拮抗薬）**：グルタミン酸の神経毒性を軽減

### せん妄 vs 認知症

| | せん妄 | 認知症（アルツハイマー） |
|--|--------|----------------------|
| 発症 | 急性 | 緩徐 |
| 意識 | 障害あり | 初期は清明 |
| 変動性 | あり | 比較的安定 |
| 可逆性 | 可逆的 | 不可逆的 |
| EEG | 非特異的異常 | 初期は正常 |

### その他の認知症
- **血管性認知症**：多発性脳梗塞による（階段状悪化が特徴）
- **レビー小体型認知症**：幻視・パーキンソン症状・抗精神病薬で悪化
- **前頭側頭型認知症（ピック病）**：人格変化・脱抑制が先行
- **クロイツフェルト・ヤコブ病**：プリオン病、急速進行性

## 2. パーソナリティ障害（Personality Disorders）

慢性で生涯にわたる固く不適切な行動・思考パターン。洞察がなく治療を求めない。

### クラスターA（奇妙・風変わり）―統合失調症スペクトラム
| 障害 | 特徴 | 覚え方 |
|------|------|-------|
| **猜疑性（Paranoid）** | 他者への不信・疑念 | Weird（奇妙） |
| **シゾイド（Schizoid）** | 社会的孤立・感情の平板化、孤独を好む | 感情欠如 |
| **統合失調型（Schizotypal）** | 奇妙な信念・思考（魔術的思考）、統合失調症と親族関係 | 魔術的思考 |

### クラスターB（劇的・感情的・衝動的）
| 障害 | 特徴 |
|------|------|
| **演技性（Histrionic）** | 注目欲求・誇張した感情表現 |
| **自己愛性（Narcissistic）** | 誇大感・共感欠如・賞賛欲求 |
| **境界性（Borderline）** | 感情不安定・分裂（splitting）・自傷・見捨てられ恐怖。治療：**DBT** |
| **反社会性（Antisocial）** | 共感欠如・法律違反・責任感なし（サイコパス）。18歳以上、素行症の既往 |

### クラスターC（不安・恐怖）
| 障害 | 特徴 |
|------|------|
| **回避性（Avoidant）** | 批判・拒絶への恐怖から社交場面を避ける（社交不安症と類似） |
| **強迫性（Obsessive-Compulsive）** | 秩序・完璧主義・コントロールへのこだわり（OCDとは別） |
| **依存性（Dependent）** | 意思決定の困難・他者への過度な依存 |

## 3. 解離症群（Dissociative Disorders）

- 記憶・意識・同一性の連続性が損なわれる
- 心理的要因（トラウマ）が背景
- 例：解離性健忘、解離性同一症（多重人格）

---

## 4. 摂食障害（Eating Disorders）

| | 神経性やせ症（AN） | 神経性過食症（BN） |
|--|------------------|-----------------|
| 体重 | 著しく低体重（BMI<17.5） | 通常体重〜過体重 |
| 行動 | 摂食制限（＋過食・排出型）| 過食＋代償行動（嘔吐・下剤） |
| 身体異常 | 徐脈・低血圧・無月経・電解質異常 | 電解質異常（低K）、歯のエナメル質損傷 |
| 死亡率 | 全精神疾患中最高 | 較低 |
| 社会的特徴 | 若い女性・高SES・先進国 | 同様 |

---

## 5. 自閉スペクトラム症（Autism Spectrum Disorder: ASD）

- コミュニケーション障害＋社会的相互作用障害＋限定的・反復的行動
- 男児に4倍多い、約1%に発症
- サヴァン症候群：突出した技能（記憶・音楽・絵画など）
- 治療：**行動療法**（唯一効果が実証された治療）

---

## 6. ADHD・素行症

| | ADHD |
|--|------|
| 症状 | 不注意・多動・衝動性（複数の状況で症状あり） |
| 年齢 | 12歳以前に症状あり |
| 男女比 | 男児に多い |
| 治療 | メチルフェニデート（リタリン）等中枢刺激薬、行動療法 |

## 学習項目への回答

### 1. 神経認知障害群について（30秒）
**定義**：神経認知障害群（Neurocognitive Disorders, NCDs）とは、**後天的に脳の機能障害により認知機能（記憶・実行機能・注意・言語・視空間・社会認知）が低下する疾患群の総称**（DSM-5）。旧「認知症」「健忘性障害」「せん妄」を統合した分類。
- **せん妄**（急性、可逆性）
- **軽度認知障害（mild NCD、MCI）**：日常生活の自立は保たれる
- **認知症（major NCD）**：日常生活の自立が損なわれる
- 原因疾患別：アルツハイマー病、血管性、レビー小体型、前頭側頭型、外傷性、HIV感染、プリオン病、パーキンソン病、ハンチントン病など

### 2. せん妄について（30秒）
**定義**：せん妄（delirium）とは、**急性に発症し変動性の経過をとる、意識・注意の障害と認知障害（記憶・見当識・言語など）を呈する症候群で、原因が身体疾患・薬剤・物質にある可逆的な精神状態**（DSM-5）。

特徴：日内変動（夕方〜夜悪化＝夕暮れ症候群）。原因（感染、薬剤、脱水、電解質異常、術後、アルコール離脱）。高齢・術後・ICU・末期患者で頻発。治療：原因治療＋環境調整＋少量ハロペリドール。**ベンゾジアゼピンは原則禁忌**（アルコール離脱と痙攣性せん妄を除く）。

### 3. アルツハイマー病について（30秒）
**定義**：アルツハイマー病（Alzheimer's disease, AD）とは、**大脳皮質と海馬を中心にアミロイドβ斑と神経原線維変化が蓄積し、神経細胞の脱落と脳萎縮を生じる、緩徐進行性・不可逆性の神経変性認知症**。認知症全体の約60〜70%を占める最多の認知症。

経過：**近時記憶障害から始まり**、失語・失認・失行・実行機能障害へ。画像：海馬萎縮・脳室拡大。病理：**アミロイドβ斑（老人斑）＋神経原線維変化（リン酸化タウ）**。リスク：加齢、女性、アポE4、ダウン症（21トリソミー）、頭部外傷、低教育。治療：AChE阻害薬（ドネペジル）、メマンチン、近年抗アミロイド抗体（レカネマブ）。

### 4. 軽度認知障害について（30秒）
**定義**：軽度認知障害（Mild Cognitive Impairment, MCI）とは、**1領域以上の認知機能低下が客観的に認められるが、日常生活の自立は保たれている、認知症の前駆段階に相当する状態**。DSM-5では「軽度神経認知障害」として独立した診断名を持つ。年間10〜15%が認知症へ移行（一部は安定または改善）。介入：運動、認知訓練、生活習慣病管理、難聴補聴、社会参加。

### 5. 血管性認知症について（30秒）
**定義**：血管性認知症（vascular dementia, VaD）とは、**脳血管障害（脳梗塞・脳出血・小血管病）による脳組織の損傷を原因とする認知症**。アルツハイマー病に次いで多い。

特徴：**階段状の悪化**、**症状のまだら**（保たれる機能と障害された機能が混在）、巣症状（麻痺・失語）、感情失禁、夜間せん妄。リスク：高血圧、糖尿病、脂質異常、心房細動、喫煙。予防が最も重要（脳卒中予防＝認知症予防）。

### 6. レビー小体型認知症について（30秒）
**定義**：レビー小体型認知症（dementia with Lewy bodies, DLB）とは、**α-シヌクレイン凝集体（レビー小体）が大脳皮質を含む広範な神経細胞内に沈着して生じる、進行性の神経変性認知症**。アルツハイマー病・血管性に次いで頻度が高い。

3徴：**①認知機能の動揺、②具体的な反復する幻視、③パーキンソニズム**。+REM睡眠行動障害、自律神経症状、転倒。**抗精神病薬（特に定型）に過敏**で錐体外路症状・悪性症候群リスク高い→使用は最小量、クエチアピン優先。AChE阻害薬は有効。

### 7. 前頭側頭型認知症（ピック病）とプリオン病による認知症（クロイツフェルト・ヤコブ病）について（30秒）
- **前頭側頭型認知症（FTD/ピック病）**：65歳未満発症が多い。**人格変化・脱抑制・常同行動**（万引き、セクハラ、過食）が**記憶障害より先行**。意味性認知症、進行性非流暢性失語などの言語型もあり。タウ・TDP-43の沈着
- **プリオン病（CJD）**：プリオン蛋白の異常凝集。**急速進行性認知症**（数ヶ月で進行）＋ミオクローヌス＋失調＋皮質盲。EEGに**周期性同期性放電（PSD）**、髄液14-3-3蛋白陽性。**致死的、治療なし**、感染対策（脳神経外科器具の処理）

### 8. パーソナリティ障害群Cluster Aについて（30秒）
**定義**：パーソナリティ障害（personality disorder）とは、**個人の文化から期待されるものから著しく偏った、内的体験と行動の持続的様式が、青年期・成人早期から始まり、長期にわたり融通性がなく、苦痛または機能障害をもたらす精神疾患**（DSM-5）。**Cluster A**は「**奇妙・風変わり群**」で、統合失調症スペクトラムと関連が深い。
- **猜疑性（妄想性）PD**：他者を信用できず疑い深い、悪意の解釈、根に持つ
- **シゾイドPD**：社会的関係を望まず孤独を好む、感情表現乏しい、性的関心低い
- **統合失調型PD**：奇妙な信念（魔術的思考、テレパシー）、奇妙な知覚体験、対人不安。統合失調症と遺伝的関連

### 9. パーソナリティ障害群Cluster Bについて（30秒）
**定義**：パーソナリティ障害**Cluster B**は、**「劇的・感情的・衝動的」な特徴を共有するパーソナリティ障害群**。対人関係と感情調節の不安定さが共通する。
- **反社会性PD**：18歳以上、15歳前から素行症の既往、法律違反、虚偽、攻撃性、無責任、後悔の欠如。男性に多い
- **境界性PD**：見捨てられ恐怖、不安定で激しい対人関係、同一性障害、衝動性、自傷・自殺企図、感情不安定、慢性的空虚感、激しい怒り、一過性解離。**DBTが第一選択**
- **演技性PD**：注意の中心でないと不快、性的誘惑的、感情の浅薄・劇場的、被影響性
- **自己愛性PD**：誇大感、賞賛欲求、共感欠如、特権意識、搾取的

### 10. パーソナリティ障害群Cluster Cについて（30秒）
**定義**：パーソナリティ障害**Cluster C**は、**「不安・恐怖」を共通特徴とするパーソナリティ障害群**。対人関係に対する不安や、自己評価の低さ、コントロールへのこだわりが特徴。
- **回避性PD**：批判・拒絶への過敏、社会的引きこもり、自己評価が低い（社交不安症と類似）
- **依存性PD**：他者への過度の依存、決定困難、別離への恐怖、見捨てられ感
- **強迫性PD**：完璧主義、規則・秩序へのこだわり、柔軟性欠如、仕事中心、ケチ、頑固。OCDとは別概念（強迫観念・行為がない）

### 11. 解離症群について（1分）
**定義**：解離症群（dissociative disorders）とは、**意識・記憶・同一性・情動・知覚・身体表象・運動制御・行動の通常の統合が破綻ないし不連続となる症状を呈する精神疾患群**（DSM-5）。多くはトラウマ（虐待、戦闘、災害）が背景。
- **解離性同一症（DID、旧多重人格）**：2つ以上の交代人格、記憶の欠落。重度の幼児期虐待が背景
- **解離性健忘**：重要な個人情報を想起できない（局所性／全般性／系統性）。**解離性遁走**：現住居から離れて新たな同一性を形成
- **離人感／現実感喪失症**：自分・周囲が現実でない・自分のものでないという持続的体験。現実検討は保たれる
- **他の特定／特定不能の解離症**：上記亜閾値
治療：トラウマ焦点化心理療法、安全性の確保、薬物は補助的。直接的な人格統合より統合的機能向上を目標

### 12. 肥満について（30秒）
**定義**：肥満（obesity）とは、**体脂肪が過剰に蓄積した状態で、健康障害を引き起こすかその恐れのある状態**（WHO）。指標として**BMI ≥25（日本基準）/ ≥30（WHO基準）**を用いる。

合併症：生活習慣病リスク↑（糖尿病、高血圧、脂質異常、心血管疾患、睡眠時無呼吸、変形性関節症、特定がん）、心理社会的問題（うつ、スティグマ、自尊感情低下）。原因：遺伝＋摂食行動＋運動不足＋環境要因（食品環境、社会経済）。治療：食事・運動・行動療法＋薬物（GLP-1作動薬）＋減量手術（BMI≥35）。

### 13. 神経性やせ症について（30秒）
**定義**：神経性やせ症（anorexia nervosa, AN）とは、**著しい低体重に至るほどの摂取エネルギー制限、体重増加・肥満への強い恐怖、自己の体重・体型に関する歪んだ認識を特徴とする摂食症**（DSM-5）。

特徴：典型的にBMI<17.5。**全精神疾患中で最も死亡率が高い**（5〜10%、自殺・心不全・電解質異常）。摂食制限型／過食排出型。徴候：徐脈、低血圧、低体温、無月経、産毛、脱水、電解質異常（低K）、骨粗鬆症。治療：栄養回復（**再栄養症候群**に注意）、家族療法（特に思春期：FBT/モーズレイ法）、CBT-E、強制入院検討。

### 14. 神経性過食症について（30秒）
**定義**：神経性過食症（bulimia nervosa, BN）とは、**反復する過食エピソード（短時間に通常より明らかに大量に食べ、制御不能感を伴う）と、体重増加を防ぐための不適切な代償行動を特徴とする摂食症**（DSM-5）。

要件：過食＋代償行動（自己誘発嘔吐、下剤・利尿薬乱用、過剰運動、絶食）が**週1回以上3ヶ月以上**。**通常は正常体重〜過体重**。徴候：耳下腺腫脹、**ラッセル徴候（手背の胼胝）**、う歯・歯のエナメル質喪失、低K血症、代謝性アルカローシス、Mallory-Weiss症候群。治療：**CBT-E（第一選択）**、対人関係療法、SSRI（特にフルオキセチン）。

### 15. 過食性障害について（30秒）
**定義**：過食性障害（binge eating disorder, BED）とは、**反復する過食エピソードがあるが、神経性過食症のような代償行動を伴わない摂食症**（DSM-5）。週1回以上3ヶ月以上。

特徴：早食い・苦しいまで食べる・空腹でなくとも・恥じて隠れて・自己嫌悪。肥満を伴うことが多い。神経性過食症より高頻度（最多の摂食障害）。治療：**CBT、対人関係療法、リスデキサンフェタミン**（米FDA承認）、SSRI、減量プログラム。

### 16. 自閉スペクトラム症について（1分）
**定義**：自閉スペクトラム症（autism spectrum disorder, ASD）とは、**①社会的コミュニケーションおよび対人的相互作用の持続的な欠陥と、②限定的・反復的な行動・興味・活動を主徴とする神経発達症**（DSM-5）。早期発達期から症状が始まる。
- **A基準（社会的コミュニケーション）**：相互的会話の困難、非言語的伝達の異常（視線・身振り）、関係性の発展・維持・理解の困難
- **B基準（限定反復行動）**：常同運動・反響言語、同一性へのこだわり、限局的興味、感覚過敏／鈍麻
- 重症度：レベル1（支援を要する）〜3（非常に十分な支援を要する）
- 知的障害・てんかん・ADHD・不安症・睡眠障害が高頻度合併
- 男児に約4倍多い、有病率約1〜2%
- 治療：**早期療育（ABA：応用行動分析、TEACCH、PECS）**、ソーシャルスキル訓練、家族支援、合併症の薬物療法（リスペリドン・アリピプラゾールが易刺激性に保険適用）

### 17. サヴァン症候群について（30秒）
**定義**：サヴァン症候群（savant syndrome）とは、**知的障害や自閉スペクトラム症など発達上の障害がありながら、特定の領域（記憶・音楽・計算・絵画など）において一般人をはるかに超える突出した能力を示す状態**。

特徴：自閉症の約10%が該当。例：カレンダー計算、電話帳暗記、一度聞いた曲を演奏、素数判定、瞬間記憶による精密描画。映画「レインマン」のモデル。中枢性統合の弱さ・大脳左右非対称な機能発達と関連。獲得性サヴァン（左前頭側頭葉障害後）も報告。

### 18. 注意欠如・多動症について（1分）
**定義**：注意欠如・多動症（attention-deficit/hyperactivity disorder, ADHD）とは、**発達水準に不相応な不注意および／または多動・衝動性が、12歳以前から複数の状況（家・学校・職場）で6ヶ月以上持続し、社会的・学業的・職業的機能を障害する神経発達症**（DSM-5）。
- **不注意**：細部への不注意、集中持続困難、聞いていないように見える、指示を完遂しない、整理が苦手、忘れ物・物の紛失、外的刺激で気が散る
- **多動・衝動性**：手足のもじもじ、座っていられない、走り回る、静かに遊べない、しゃべりすぎる、答えを待てない、順番を待てない、他人を遮る
- 男児に多い（女児は不注意優勢で見逃されやすい）。成人期にも約60%で症状残存
- 高頻度合併：学習症、ASD、素行症、不安症、うつ病、物質乱用、二次的自尊感情低下
- 治療：**心理教育＋環境調整＋ペアレントトレーニング＋行動療法＋薬物療法**

### 19. 注意欠如・多動症の薬物療法について（30秒）
- **中枢刺激薬（第一選択）**：
  - **メチルフェニデート徐放剤（コンサータ）**：DA・NA再取り込み阻害
  - **リスデキサンフェタミン（ビバンセ）**：プロドラッグ、6歳以上小児
  - 副作用：食欲低下、不眠、頭痛、心拍・血圧↑、成長抑制、tic悪化、依存性（処方は登録医のみ）
- **非中枢刺激薬**：
  - **アトモキセチン（ストラテラ）**：選択的NA再取り込み阻害
  - **グアンファシン（インチュニブ）**：α2A作動薬
  - 効果発現は中枢刺激薬より遅いが依存性なし
- **ブプロピオン**：適応外だが成人ADHDに使用される

### 20. 素行症、反抗挑発症について（30秒）
**定義**：
- **反抗挑発症（oppositional defiant disorder, ODD）**：**怒り・易刺激性、口論的・挑発的行動、執念深さを特徴とする情緒・行動の障害**。権威者への反抗が中心、重大な権利侵害は少ない。**素行症の前駆**となることがある
- **素行症（conduct disorder, CD）**：**他者の基本的権利、または年齢相応の社会的規範や規則を侵害する反復的・持続的な行動パターン**（攻撃、財産破壊、虚偽・窃盗、重大な規則違反）。**18歳未満**で発症（小児発症型／青年発症型）。**18歳以上で反社会性パーソナリティ障害**へ移行することがある
- 治療：**ペアレントトレーニング**、家族療法、マルチシステム療法（MST）、CBT、ADHDなど併存症の薬物療法

### 21. トゥレット症について（30秒）
**定義**：トゥレット症（Tourette syndrome / Tourette's disorder）とは、**多発性運動チックと1つ以上の音声チックが、ともに18歳以前に発症し、1年以上持続するチック症**（DSM-5）。チック＝突発的・反復的・非律動的な運動／発声。前駆衝動（ムズムズ感）あり。約半数で**ADHD・OCD・LD・不安・うつ**の合併。学童期に最強、思春期以降軽快も多い。治療：心理教育、**Habit Reversal Training（HRT）**、薬物（リスペリドン、アリピプラゾール、グアンファシン、クロニジン）。

### 22. 分離不安症について（30秒）
**定義**：分離不安症（separation anxiety disorder）とは、**愛着対象（多くは親）からの分離、または分離が起こりうるという予期に対して、発達段階に不釣り合いな過剰な恐怖・不安を示す不安症**（DSM-5）。

症状：学校拒否、独りで寝られない、悪夢、身体症状（頭痛・腹痛）。**小児は4週間以上、成人は6ヶ月以上**。学齢期女児に多い。経過：登校不能、社会的引きこもり。治療：**CBT（曝露中心、家族介入）**、SSRIは中等症以上に。

### 23. 排泄症群について（30秒）
**定義**：排泄症群（elimination disorders）とは、**膀胱・腸の制御を獲得する発達年齢に達した児が、不適切な場所で排尿または排便を反復する精神疾患群**（DSM-5）。**遺尿症**と**遺糞症**を含む。

- **遺尿症（enuresis）**：5歳以上で、不随意または意図的な尿失禁。**夜尿型／昼尿型／混合型**。3ヶ月以上週2回以上、または苦痛を伴う。器質的疾患・薬剤を除外。治療：**行動療法（夜尿アラーム、報酬法、水分制限）**、デスモプレシン（ADH類似薬）、TCA（イミプラミン）
- **遺糞症（encopresis）**：4歳以上で、不適切な場所での排便（不随意または意図的）。**便秘・溢流性失禁あり／なし**で分類。器質的疾患を除外。治療：**便秘の管理（緩下剤）＋行動療法（排便スケジュール、報酬法）＋家族支援**
