## 🎴 ① まずはAnkiデッキ（391問・一番おすすめ）

BRS各章の**章末Review Testを全問**、問題＋日本語の解答解説つきでAnkiカードにしてあります。
表面＝BRSの英文問題＋選択肢／裏面＝正答＋解説（表・太字つき）。章ごとにサブデッキ分けです。

### 使い方（3ステップ）

1. **Ankiを入れる**（無料）… 公式 → https://apps.ankiweb.net
   - スマホアプリ「AnkiMobile(iOS有料)／AnkiDroid(Android無料)」でも、PCで取り込んだ後に同期すれば使えます。
2. **デッキファイルをダウンロード**
   - このページ上部の **`行動科学BRS_全章391問.apkg`** をクリック → 右上の **ダウンロード（⤓）ボタン**で保存。
3. **Ankiで取り込む**
   - Anki を開く → メニュー **ファイル → 読み込む（Import）** → 保存した `.apkg` を選ぶ → **「インポート」**を押す。
   - （`.apkg` をダブルクリックでもOK）

完了すると、左の一覧に **`行動科学BRS`** が現れ、`Ch01 生涯発達1` … `Ch24 法と医療` のサブデッキに分かれて入ります。

> 💡 **更新版が出たら**：新しい `.apkg` を同じ手順で読み込めばOK。カードIDを固定してあるので、**重複せず中身だけ上書き更新**されます（学習記録は消えません）。

## ⬇️ ダウンロードのしかた（GitHub初めてでもOK）

### A. 全部まとめて欲しい（ZIP）
1. このページ上部の緑色の **`< > Code`** ボタンをクリック
2. **「Download ZIP」** を選ぶ → 解凍すれば全ファイルが手元に入ります。

### B. 1ファイルだけ欲しい（例：Ankiデッキ、ある章のノート）
1. 欲しいファイル名をクリックして開く
2. 右上の **ダウンロード（⤓）アイコン**（または「Raw」を右クリック→保存）でその1ファイルだけ保存。

> 🪟 **Windowsの人へ**：フォルダ名に `:`（コロン）が含まれるため、ZIP全体の解凍で一部フォルダが展開できないことがあります。その場合は **欲しいファイルを1つずつ（方法B）** か、**Ankiデッキだけ** を取得してください（Ankiデッキ単体なら問題なく使えます）。
