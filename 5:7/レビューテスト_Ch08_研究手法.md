### 脳画像検査・脳波の使い分け（最重要）
| 検査 | 適応 |
|------|------|
| **CT** | **構造的**脳変化（アルツハイマー：脳室拡大、出血、腫瘍）|
| 構造MRI | 脱髄疾患（MS）、被曝なし、軟部組織 |
| **PET** | **脳の機能的活動**（グルコース代謝、特定タスク中の活動）|
| **fMRI** | 脳機能活動（PETに類似、被曝なし）|
| SPECT | PET/fMRI代替、ガンマカメラで簡便 |
| **EEG** | **てんかん、せん妄 vs 認知症**の鑑別 |
| **誘発EEG**（evoked）| **乳幼児の聴覚・視覚評価**、脳死判定 |

### 特殊な検査法
| 検査 | 使用場面 |
|------|-------|
| **乳酸ナトリウム静注**（sodium lactate）| **パニック症の誘発**（診断確認）|
| **アモバルビタール面接**（Amytal）| **変換症**（身体症状の心因性評価）、精神病患者 |
| **DST**（デキサメタゾン抑制試験）| **大うつ病の約50%で陽性**（抑制されない）|
| **ガルバニック皮膚反応** | 嘘発見器（皮膚抵抗の変化）|

### 内分泌と精神症状
| 疾患 | 身体所見 | 精神症状 |
|------|--------|-------|
| **甲状腺機能亢進**（Graves病）| 眼球突出、首腫瘤、体重減少、動悸、発熱 | **不安**・不眠 |
| **甲状腺機能低下**| 倦怠、体重増加、声低下、浮腫 | **うつ** |
| **アジソン病**（副腎機能低下）| **皮膚色素沈着**（しわ・口腔）| うつ、混乱、精神病 |
| **クッシング症候群**（副腎機能亢進）| 満月様顔貌、紫紋、水牛肩 | うつ、不安、躁 |
| **急性間欠性ポルフィリン症** | **紫赤色尿**（ポルフォビリノーゲン↑）、腹痛 | **妄想**、幻覚、うつ |

### IQと知的能力障害の分類
```
IQ = (精神年齢 MA / 生活年齢 CA) × 100
```
| IQ | 分類 |
|----|------|
| ≥130 | 優秀 |
| 90〜109 | **平均** |
| 70〜89 | 境界線〜平均以下 |
| 50〜70 | **軽度知的能力障害** |
| 35〜55 | 中等度 |
| 20〜40 | 重度 |
| <20 | 最重度 |

### 代表的な心理検査
| 検査 | 目的 |
|------|------|
| **Folstein MMSE** | 認知機能スクリーニング（30点満点、**18点以下＝認知障害**、23点以上＝正常）|
| グラスゴー昏睡スケール | 意識レベル（3〜15点）|
| **ヴァインランド社会成熟度尺度** | **知的障害者の日常生活スキル**評価（グループホーム入所評価）|
| **MMPI-2** | **客観的**人格検査、true/false、うつ・パラノイア・統合失調等を評価 |
| **ロールシャッハ** | **投影**法、インクのしみ、防衛機制の同定 |
| **TAT** | 投影法、絵の物語から無意識を探る |
| **ベックうつ評価（BDI）** | **自己評価**のうつ尺度 |
| ハミルトンうつ・ラスキン評価 | **検査者による**うつ評価 |
| **ウィスコンシンカードソーティング** | **実行機能・抽象思考**（前頭葉）|
| ベンダー視覚運動ゲシュタルト | **視覚運動能力**（脳卒中後の模写困難）|

### MSE（精神状態検査）の構成要素
- 外見・行動・面接者への態度
- 意識レベル（GCS）
- 認知（見当識・記憶・注意・空間能力）
- 気分と感情
- 思考（形式・内容：妄想）
- 知覚（**幻覚・錯覚**）
- 判断力・病識

### 知覚・思考の障害
| 用語 | 定義 |
|------|------|
| **幻覚（hallucination）**| **知覚**の障害。存在しないものを知覚 |
| 錯覚（illusion）| 現実を誤知覚 |
| **妄想（delusion）**| **思考内容**の障害。誤った固い信念 |

### 向精神薬の副作用モニタリング
| 薬 | モニタリング項目 |
|----|--------------|
| **クロザピン**（Clozaril）| **無顆粒球症**（WBC<2000）→ 定期血球検査必須 |
| **リチウム** | 血中濃度（治療域0.6〜1.2 mEq/L）、腎機能、甲状腺機能 |
| **カルバマゼピン、バルプロ酸** | **肝機能**、血球検査 |

## 問題1（Ch8 Q1）IQ計算と知的能力障害

A 12-year-old child who is having difficulty in school is given an intelligence test. The test determines that the child is functioning mentally at the level of an 8-year-old child.

(A) Severe intellectual disability  (B) Moderate intellectual disability  (C) Mild intellectual disability  (D) Borderline  (E) Average

**【正答】** (C) Mild intellectual disability

**【解説】**
**IQ計算：**
```
IQ = (MA / CA) × 100 = (8 / 12) × 100 ≈ 66
```

**IQ 66 → 軽度知的能力障害（IQ 50〜70）**

**IQ分類（暗記）：**
| IQ | 分類 |
|----|------|
| 90〜109 | 平均 |
| 71〜89 | 境界〜平均以下 |
| **50〜70** | **軽度**知的障害 |
| 35〜55 | 中等度 |
| 20〜40 | 重度 |
| <20 | 最重度 |

**類題（Ch8 Q2）：**
MA=12, CA=10 → IQ = 120 → 優秀

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題2（Ch8 Q2）IQ計算（MA 12 / CA 10）

A child is tested and found to have a mental age of 12 years. The child's chronological age is 10 years. What is the IQ of this child?

(A) 40  (B) 60  (C) 80  (D) 100  (E) 120

**【正答】** (E) 120

**【解説】**
**IQ計算の公式：**
```
IQ = (MA / CA) × 100 = (12 / 10) × 100 = 120
```

**IQ 120 → 平均より上（above average）**

**IQ分類（暗記必須）：**
| IQ | 分類 |
|----|------|
| ≥130 | 優秀（gifted）|
| 110〜129 | 平均より上 |
| **90〜109** | **平均（average）** |
| 71〜89 | 境界線〜平均以下 |
| 50〜70 | 軽度知的能力障害 |
| 35〜55 | 中等度 |
| 20〜40 | 重度 |
| <20 | 最重度 |

**類題（Ch8 Q1）との対比：**
- Q1: MA=8, CA=12 → IQ ≈ 66 → **軽度知的障害**
- Q2: MA=12, CA=10 → IQ = 120 → **平均以上**

**ポイント：**
- 精神年齢（MA）が生活年齢（CA）より上 → IQ > 100
- 同年齢の平均より知的に進んでいる
- ただし、**成人ではこの式は通常使わず、偏差IQ**（標準偏差ベース、平均100、SD 15）を用いる

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題3（Ch8 Q3）幻聴は知覚の障害

A 29-year-old woman tells the doctor that she often hears the voice of Abraham Lincoln speaking directly to her. This woman is showing a disorder of

(A) Perception  (B) Insight  (C) Judgment  (D) Mood  (E) Affect

**【正答】** (A) Perception

**【解説】**
**覚えるべきルール：**
- **幻覚（hallucination）= 知覚（perception）の障害**
- 存在しない刺激を知覚する
- 本問は**幻聴（auditory hallucination）**：リンカーンの声が聞こえる

**MSE（精神状態検査）で評価される項目：**
| 項目 | 内容 |
|------|------|
| **知覚（perception）** | **幻覚・錯覚** |
| 思考（thought）| 妄想（思考内容）、連合弛緩（思考形式）|
| 病識（insight）| 自分の病気への理解 |
| 判断力（judgment）| 状況判断・意思決定能力 |
| 気分（mood）| 持続的な感情状態（患者の主観）|
| 感情（affect）| 観察される感情表現 |

**妄想と幻覚の対比（重要）：**
- 「CIAが自分を監視している」と信じる → **妄想**（思考内容）
- 「CIAの声が聞こえる」 → **幻聴**（知覚）
- 本問はリンカーンの**声を聞く** → 幻聴 → 知覚障害

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題4（Ch8 Q4）IQ 90の分類

A child is tested and is found to have an IQ of 90. What category of intellectual function best describes this child?

(A) Severely intellectually disabled  (B) Moderately intellectually disabled  (C) Mildly intellectually disabled  (D) Borderline  (E) Average

**【正答】** (E) Average

**【解説】**
**覚えるべきルール：**
- **IQ 90〜109 → 平均（average）**
- 本問のIQ 90 → 平均範囲の下端だが平均

**IQ分類（再掲）：**
| IQ | 分類 |
|----|------|
| ≥130 | 優秀 |
| 110〜129 | 平均以上 |
| **90〜109** | **平均** |
| **71〜89** | **境界（borderline）〜平均以下** |
| 50〜70 | 軽度知的障害 |
| 35〜55 | 中等度知的障害 |
| 20〜40 | 重度知的障害 |
| <20 | 最重度知的障害 |

**「境界」（borderline）の重要性：**
- IQ 71〜84は「境界知能」
- 知的障害の診断基準（DSM-5）はIQ約70未満
- 境界知能は知的障害ではないが、学業・就労で配慮が必要なことも

**知能検査の代表：**
- **WAIS-IV**：成人（16歳以上）
- **WISC-V**：児童（6〜16歳）
- **スタンフォード-ビネー**：2歳〜成人
- 知能は **言語性IQ + 動作性IQ** などで評価

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題5（Ch8 Q5）MMPI-2の評価対象

A doctor is evaluating a 20-year-old female patient. Which of the following characteristics of the patient is best evaluated using the Minnesota Multiphasic Personality Inventory-2 (MMPI-2)?

(A) Skills for daily living  (B) Depression  (C) Knowledge of general information  (D) Reading comprehension  (E) Intelligence

**【正答】** (B) Depression

**【解説】**
**覚えるべきルール：**
- **MMPI-2**：**客観的人格検査**（true/false形式、約567問）
- 臨床尺度で評価する内容：
  - **うつ**（Depression）
  - 心気症（illness anxiety disorder）
  - パラノイア
  - 統合失調症
  - 精神病質、ヒステリー、性役割など
- プライマリケア医も使用可能（特別なトレーニング不要）

**他の選択肢の対応する検査：**
| 評価したい内容 | 適切な検査 |
|---------------|----------|
| 日常生活スキル | **ヴァインランド社会成熟度尺度** |
| **うつ** | **MMPI-2**、BDI-II（自己評価）、ラスキン尺度（評価者）|
| 一般知識・読解 | **WAIS（成人）**, WISC-R（小児）|
| 知能 | WAIS, WISC, スタンフォード-ビネー |

**MMPI-2 vs 投影法：**
- **MMPI-2**：客観的、true/false、特別なトレーニング不要
- **ロールシャッハ・TAT**：投影法、解釈に専門的訓練必要

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題6（Ch8 Q6）IQ 50の6歳児に期待される能力

A 6-year-old child is tested and is found to have an IQ of 50. There are no significant medical findings. At this time, it can be expected that the child is able to

(A) identify some colors  (B) ride a two-wheeled bicycle  (C) understand that death is permanent  (D) copy a triangle  (E) utilize an internalized moral sense of right and wrong

**【正答】** (A) identify some colors

**【解説】**
**精神年齢の計算：**
```
IQ = MA/CA × 100
50 = MA/6 × 100
MA = 3歳
```
- 6歳でIQ 50 → 精神年齢は **3歳相当**
- → 3歳児に可能なことが期待される

**3歳児の発達（暗記必須）：**
- **色の識別**ができる
- 円を描く（copy a circle）
- 短い文を話す
- 自分の名前を言える

**6歳前後で獲得される能力（本問では未獲得）：**
| 能力 | 獲得時期の目安 |
|------|-------------|
| 二輪自転車に乗る | 6歳ごろ |
| 死は永続的と理解 | 6歳ごろ（具体的操作期）|
| 三角形を模写 | 6歳ごろ |
| 内在化された道徳感 | 6歳〜（前操作期 → 具体的操作期で確立）|

**Piaget発達段階との対応：**
| 段階 | 年齢 | 特徴 |
|------|-----|------|
| 感覚運動期 | 0〜2歳 | 対象の永続性 |
| 前操作期 | 2〜7歳 | 自己中心的、ごっこ遊び、3歳で**色の識別** |
| 具体的操作期 | 7〜11歳 | 保存性、論理的、**死の理解** |
| 形式的操作期 | 11歳〜 | 抽象的思考 |

**死の概念の発達：**
- **2歳未満**：理解しない
- **2〜6歳**：可逆的な「眠り」と捉える
- **6歳以降**：死は永続的・不可避と理解

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題8（Ch8 Q8）ロールシャッハ：投影法と防衛機制

For determining, using bilaterally symmetrical inkblots, which defense mechanisms are used by a 25-year-old woman, what is the most appropriate test?

(A) TAT  (B) MMPI-2  (C) WISC-R  (D) Rorschach Test  (E) Vineland Social Maturity Scale  (F) WRAT  (G) BDI-II  (H) Raskin Depression Scale  (I) Wisconsin Card Sorting Test

**【正答】** (D) Rorschach Test

**【解説】**
**覚えるべきルール：**
- **ロールシャッハ・テスト（Rorschach Test）**
  - **左右対称のインクのしみ（inkblots）** 10枚を見せる
  - 患者がそこに何を見るかを語らせる
  - **投影法（projective test）** の代表
  - **防衛機制**・思考障害・無意識の葛藤を評価
  - 解釈には**専門的訓練**が必要

**投影法 vs 客観法：**
| 種類 | 代表 | 特徴 |
|------|------|------|
| **投影法** | **ロールシャッハ**（インクのしみ）、**TAT**（絵から物語）| 無意識・防衛機制を評価、解釈に訓練必要 |
| **客観法** | **MMPI-2**（true/false 567問）| 性格特徴を客観的に数値化、特別な訓練不要 |

**TATとの違い：**
- **ロールシャッハ**：インクのしみ → 知覚・連想 → 防衛機制
- **TAT（Thematic Apperception Test）**：曖昧な絵 → 物語を作らせる → 無意識の欲求・葛藤・対人関係

**他の選択肢の用途：**
- WISC-R：児童の知能
- ヴァインランド：知的障害者の日常生活スキル
- WRAT：学力（読み書き計算）
- BDI-II：うつ自己評価
- ラスキン尺度：うつ評価者評価
- WCST：実行機能・抽象思考

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題9（Ch8 Q9）うつ病の自己評価尺度（BDI-II）

For evaluating depression in a 54-year-old male patient using a self-rating scale, what is the most appropriate test?

(A) TAT  (B) MMPI-2  (C) WISC-R  (D) Rorschach Test  (E) Vineland Social Maturity Scale  (F) WRAT  (G) BDI-II  (H) Raskin Depression Scale  (I) Wisconsin Card Sorting Test

**【正答】** (G) BDI-II

**【解説】**
**覚えるべきルール：**
- **Beck Depression Inventory-II（BDI-II）**：**患者自身が記入する**うつ病の自己評価尺度
- 21項目、各0〜3点、合計63点満点
- **自己評価（self-rating）** がキーワード

**自己評価 vs 評価者評価（重要）：**
| 検査 | 評価者 |
|------|-------|
| **BDI-II（Beck）** | **患者自身（self-rating）** |
| Zung自己評価うつ尺度 | 患者自身 |
| **ハミルトンうつ評価尺度（HAM-D）** | **検査者** |
| **ラスキンうつ尺度** | **検査者** |

**BDI-IIスコアの解釈：**
| スコア | 解釈 |
|--------|------|
| 0〜13 | 最小限のうつ |
| 14〜19 | 軽度のうつ |
| 20〜28 | 中等度のうつ |
| 29〜63 | **重度のうつ** |

- スコア60前後 → **重度のうつ・無快感症（anhedonia）**

**MMPI-2でもうつは評価できるが、本問では「self-rating」が指定 → BDI-II一択**

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題10（Ch8 Q10）プライマリケアでのMMPI-2活用

For evaluating, by a primary care physician, illness anxiety disorder in a 54-year-old male patient using true/false questions, what is the most appropriate test?

(A) TAT  (B) MMPI-2  (C) WISC-R  (D) Rorschach Test  (E) Vineland Social Maturity Scale  (F) WRAT  (G) BDI-II  (H) Raskin Depression Scale  (I) Wisconsin Card Sorting Test

**【正答】** (B) MMPI-2

**【解説】**
**覚えるべきルール：**
- **MMPI-2（Minnesota Multiphasic Personality Inventory-2）**
  - **true/false 形式**（約567問）
  - **客観的人格検査**
  - **特別なトレーニング不要** → プライマリケア医も使用可能
  - 評価項目：うつ、**心気症（illness anxiety disorder）**、パラノイア、統合失調症、ヒステリー、精神病質、性役割など

**本問のキー（3つの条件すべてを満たすのはMMPI-2のみ）：**
1. プライマリケア医が使う → 特別なトレーニング不要 → **MMPI-2**（投影法は除外）
2. true/false形式 → **MMPI-2**
3. 心気症（illness anxiety disorder）の評価 → MMPI-2の臨床尺度に含まれる

**他の選択肢の除外：**
- TAT・ロールシャッハ → 投影法、専門訓練必要 → 除外
- WISC-R → 児童知能 → 除外
- ヴァインランド → 日常生活スキル → 除外
- WRAT → 学力 → 除外
- BDI-II・ラスキン → うつ評価のみ → 除外
- WCST → 実行機能 → 除外

**MMPI-2の特徴のまとめ：**
- 客観的（objective）
- self-report（自己報告型）
- 妥当性尺度を含む（嘘の検出）
- 567の質問にtrue/falseで答える
- 心気症・うつ・ヒステリー・精神病質・男女性・パラノイア・精神衰弱・統合失調症・軽躁・社会的内向性 などを評価

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題11（Ch8 Q11）抽象的思考と実行機能

The most appropriate test for evaluating abstract reasoning and problem solving in a 54-year-old female patient is the

(A) TAT  (B) MMPI-2  (C) WISC-R  (D) Rorschach Test  (E) Vineland Social Maturity Scale  (F) WRAT  (G) BDI-II  (H) Raskin Depression Scale  (I) Wisconsin Card Sorting Test

**【正答】** (I) Wisconsin Card Sorting Test

**【解説】**
**覚えるべきルール：**
- **ウィスコンシンカードソーティングテスト（WCST）**：**抽象的思考・問題解決能力**を評価
- 128枚のカードを色・形・数で分類させる
- 分類ルールは途中で変更され、患者がそれに**柔軟に適応できるか**を見る
- **前頭葉機能（実行機能）**の評価
- 統合失調症、前頭葉病変で成績低下

**他の選択肢の用途まとめ：**
| 検査 | 主な評価対象 |
|------|------------|
| TAT | 投影法（無意識・物語）|
| MMPI-2 | 客観的人格検査 |
| WISC-R | 小児の知能 |
| ロールシャッハ | 投影法（インクのしみ）|
| ヴァインランド | 知的障害者の日常生活 |
| WRAT | 学力（読み書き計算）|
| BDI-II | うつ自己評価 |
| ラスキン尺度 | うつ評価者評価 |
| **WCST** | **実行機能・抽象思考**（前頭葉）|

**臨床的意義：**
- 統合失調症患者では前頭葉機能低下を反映してWCST成績が低下
- 認知症の評価にも応用される

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 問題12（Ch8 Q12）妄想の定義

A 24-year-old patient with schizophrenia tells the physician that the Central Intelligence Agency (CIA) is listening to his telephone conversations through his television set.

(A) A hallucination  (B) An illusion  (C) Clouding of consciousness  (D) Blunted affect  (E) A delusion

**【正答】** (E) A delusion

**【解説】**
**覚えるべき定義の違い：**
| 用語 | 定義 | 例 |
|------|------|---|
| **妄想（Delusion）**| **思考内容**の障害、**誤った固い信念**（本問）| 「CIAが監視している」|
| **幻覚（Hallucination）**| **知覚**の障害、**存在しないもの**を知覚 | 誰もいないのに声が聞こえる |
| **錯覚（Illusion）**| 現実の**誤知覚**（本当にあるものを別のものと思う）| 木が人影に見える |
| 意識混濁 | 外的事象への反応不能 | 傾眠・せん妄 |
| **感情鈍麻（Blunted affect）**| 感情表現の著減 | 統合失調症の陰性症状 |

**本問のキー：**
「CIAが監視」は**信念**（思考） → **妄想**
もし「CIAの声が聞こえる」なら**幻覚**（幻聴）

**【担当グループ】** グループ20　（BRS 第8版 Ch8）

## 【問題を解くための最重要まとめ】

### 検査選択の一発判別
- 構造的脳変化 → **CT**
- 機能的脳活動 → **PET/fMRI**
- せん妄 vs 認知症 → **EEG**
- 乳児の聴覚 → **誘発EEG**
- パニック症の誘発 → **乳酸ナトリウム**
- 変換症の評価 → **アモバルビタール**

### 内分泌と精神症状の対応
- 甲状腺**亢進** → **不安**
- 甲状腺**低下** → **うつ**
- アジソン病（コルチゾール↓）→ **皮膚色素沈着**＋うつ
- クッシング（コルチゾール↑）→ 満月様顔貌＋うつ・躁
- 急性ポルフィリン症 → **紫赤色尿**＋妄想

### 心理検査の選択
- 認知機能スクリーニング → **MMSE**（30点満点、23点以上正常、18点以下障害）
- 知的障害者の日常生活 → **ヴァインランド尺度**
- 客観的人格検査 → **MMPI-2**（true/false）
- 投影法（無意識） → **ロールシャッハ**（インクのしみ）、**TAT**（絵の物語）
- うつ病自己評価 → **BDI**
- 実行機能 → **ウィスコンシンカードソーティング**

### 薬物モニタリング
- **クロザピン** → **無顆粒球症**（WBC<2000で中止）
- リチウム → 血中濃度、腎、甲状腺
- カルバマゼピン・バルプロ酸 → 肝機能、血球

### 用語の区別
- **妄想** = 思考内容の障害（誤った信念）
- **幻覚** = 知覚の障害（存在しないもの）
- **錯覚** = 誤知覚（あるものを別物と）

### IQの計算と分類
- IQ = MA/CA × 100
- 50〜70 → 軽度知的障害
- 90〜109 → 平均
