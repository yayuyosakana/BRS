## 1. 遺伝と行動

### 双子研究
- **一卵性双生児（MZ）**：ほぼ同一ゲノム → 一致率（concordance rate）が高ければ遺伝因の関与大
- **二卵性双生児（DZ）**：平均50%のゲノム共有 → 兄弟姉妹と同等
- 一致率 MZ > DZ：遺伝因の存在を支持
- 例：統合失調症（MZ約50%、DZ約15%）

### 発症に遺伝因が関わる精神疾患
- 統合失調症、双極性障害、自閉スペクトラム症、ADHD、アルツハイマー病（初老期発症）

## 4. 辺縁系（Limbic System）

感情・記憶・動機付けに関与。

### 海馬（Hippocampus）
- **短期記憶→長期記憶の変換**（陳述記憶・エピソード記憶）
- アルツハイマー病で最初に萎縮する領域
- 海馬両側障害 → 新しい記憶が形成できない（前向性健忘）

### 扁桃体（Amygdala）
- **恐怖・攻撃性の感情処理**
- 感情的意味のある感覚刺激を評価
- 両側障害 → **クリューバー・ビューシー症候群**（従順化・過性欲・口性欲）

## 7. 主要な神経伝達物質

### ドーパミン（Dopamine）
- 産生：黒質（nigrostriatal tract）、腹側被蓋野（mesolimbic / mesocortical tract）
- 機能：運動調節、報酬・動機付け、認知・実行機能
- **増加**：統合失調症（陽性症状）← 過活動（中脳辺縁系路）
- **減少**：パーキンソン病（運動障害）、統合失調症陰性症状（中脳皮質路低活動）、うつ病
- 代謝物：**HVA（ホモバニリン酸）**

### ノルアドレナリン（Norepinephrine / Noradrenaline）
- 産生：青斑核（locus ceruleus）
- 機能：覚醒、注意、ストレス反応、気分
- **増加**：不安症、躁状態
- **減少**：うつ病
- 代謝物：**MHPG（うつ病で低下）、VMA（褐色細胞腫で高値）**

### セロトニン（Serotonin）
- 産生：縫線核（raphe nuclei）
- 機能：気分、衝動性、睡眠、食欲
- **減少**：うつ病、不安症、強迫症、攻撃性・衝動性↑、自殺
- 代謝物：**5-HIAA（攻撃性・衝動性で低下）**

### アセチルコリン（Acetylcholine: ACh）
- 産生：マイネルト基底核（nucleus basalis of Meynert）
- 機能：記憶・学習、筋収縮、副交感神経節後繊維
- **減少**：アルツハイマー病（記憶障害）
- グルタミン酸も両方に関与（アルツハイマー＆統合失調症）

### GABA（γ-アミノ酪酸）
- 主要な**抑制性**神経伝達物質
- **減少**：不安症（GABAの活性低下 → 不安↑）、てんかん
- ベンゾジアゼピン・バルビツレート → GABA-A受容体を強化（抗不安効果）

### グルタミン酸（Glutamate）
- 主要な**興奮性**神経伝達物質
- アルツハイマー病と統合失調症の両方に関与
- 過剰活性 → 神経毒性 → 神経細胞死
- メマンチン（Memantine）= NMDA受容体拮抗薬（グルタミン酸の過剰を阻害）

## 学習項目への回答

### 1. 双子研究（30秒）
**定義**：双子研究（twin study）とは、**一卵性双生児（MZ）と二卵性双生児（DZ）の形質一致率を比較することで、ある形質に対する遺伝因と環境因の寄与を推定する行動遺伝学の研究法**。
- **一卵性双生児（MZ）**：ゲノムほぼ100%共有
- **二卵性双生児（DZ）**：兄弟姉妹と同じ約50%共有
- 一致率（concordance）がMZ＞DZなら遺伝因の関与を示唆
- 養子研究・家系研究と組み合わせて遺伝率を推定
- 例：統合失調症（MZ約50%／DZ約15%）、双極症（MZ約70%）、自閉症（MZ約60〜90%）

### 2. 行動症状を持つ精神疾患と器質性疾患の違い（30秒）
**定義**：
- **器質性疾患（organic disease）**：**脳・身体の構造的または生化学的異常が画像・検査で明確に同定できる疾患**（脳腫瘍、認知症、甲状腺機能異常、てんかん、薬物中毒など）
- **機能性精神疾患（functional psychiatric disorder）**：**明確な器質的異常が同定できず、診断は症候学的（DSM/ICD基準）に行う精神疾患**（うつ病、統合失調症、不安症など）。実際には神経伝達物質・神経回路レベルの「器質的」異常が想定されるが、ルーチン検査では検出困難
- 治療：器質性は原因治療が第一、機能性は薬物＋心理社会的介入

### 3. 神経系の分類と概要（1分）
**定義**：神経系（nervous system）とは、**個体内外の刺激を感受し、情報を統合・処理して、効果器に出力するネットワーク**。中枢神経系（CNS）と末梢神経系（PNS）に大別され、PNSはさらに体性神経系と自律神経系に分かれる。

```
神経系
├ 中枢神経系（CNS）
│  ├ 脳：大脳（皮質・基底核・辺縁系）、間脳（視床・視床下部）、脳幹（中脳・橋・延髄）、小脳
│  └ 脊髄
└ 末梢神経系（PNS）
   ├ 体性神経系：感覚神経・運動神経（随意）
   └ 自律神経系（不随意）
      ├ 交感神経系：闘争・逃走（fight or flight）
      └ 副交感神経系：休息・消化（rest and digest）
```

### 4. 前頭葉（1分）
- **(1)機能**：実行機能（計画・判断・抑制）、運動（一次運動野）、ブローカ野（左：言語産生）、人格・社会行動、ワーキングメモリ
- **(2)障害症状**：
  - 背外側部障害：実行機能障害（保続、計画立案不能）、無為
  - 眼窩部障害：脱抑制、衝動性、社会的不適切行動（フィネアス・ゲージの例）
  - ブローカ野障害：運動性失語（理解可、産生不可、たどたどしい発話）
  - 一次運動野障害：対側半身麻痺
  - 前頭葉認知症（前頭側頭型認知症、ピック病）：人格変化・脱抑制

### 5. 側頭葉（1分）
- **(1)機能**：聴覚（一次聴覚野）、言語理解（左：ウェルニッケ野）、記憶（海馬）、感情（扁桃体）、顔認識（紡錘状回）
- **(2)障害症状**：
  - ウェルニッケ野障害：感覚性失語（流暢だが意味不明、理解不能）
  - 海馬障害：前向性健忘（新規記憶形成不能、HM症例）
  - 扁桃体両側障害：**クリューバー・ビューシー症候群**（過従順、口唇傾向、過性欲、視覚失認）
  - 側頭葉てんかん：複雑部分発作、自動症、デジャヴュ、嗅覚幻覚
  - 紡錘状回障害：相貌失認

### 6. 後頭葉（1分）
- **(1)機能**：視覚情報処理（一次視覚野V1、視覚連合野V2-V5）
- **(2)障害症状**：
  - 一次視覚野障害：**皮質盲**（眼は正常でも見えない）、対側同名半盲
  - 視覚連合野障害：視覚失認（見えるが何かわからない）、色覚失認
  - **アントン症候群**：皮質盲を否認する（病態失認）
  - **バリント症候群**（両側頭頂後頭葉障害）：視覚性運動失調、精神性注視麻痺、同時失認

### 7. 頭頂葉（1分）
- **(1)機能**：体性感覚（一次体性感覚野）、空間認識、感覚統合、計算、書字
- **(2)障害症状**：
  - 一次体性感覚野障害：対側の感覚障害
  - **右（劣位）頭頂葉障害**：左半側空間無視、構成失行、着衣失行
  - **左（優位）頭頂葉障害**：**ゲルストマン症候群**（失算、失書、手指失認、左右失認）
  - 観念運動失行：道具使用の困難

### 8. 辺縁系（海馬と扁桃体）（1分）
**定義**：辺縁系（limbic system）とは、**大脳半球の内側面で大脳新皮質と脳幹の間に位置し、情動・記憶・動機付け・自律機能を担う一連の脳構造の総称**（海馬、扁桃体、帯状回、視床下部、中隔核、嗅球などを含む）。Papezの回路、Yakovlevの回路として情動回路を構成。
- **海馬**：
  - (1)短期記憶→長期記憶への変換、空間記憶、エピソード記憶
  - (2)両側障害→前向性健忘（HM症例、コルサコフ症候群）。アルツハイマー病で**最初に萎縮**する領域
- **扁桃体**：
  - (1)恐怖・怒りなどの情動処理、情動的記憶の形成、社会的情動の評価
  - (2)両側障害→**クリューバー・ビューシー症候群**（恐怖反応消失、過従順、口唇傾向、過性欲）。PTSDで過活動、自閉症で機能異常

### 9. 記憶の分類と関係する脳領域（1分）
**定義**：記憶（memory）とは、**情報を符号化（encoding）→貯蔵（storage）→想起（retrieval）する脳の機能**。保持時間（感覚記憶／短期／長期）、意識的想起の有無（陳述／非陳述）、内容（エピソード／意味／手続き）などで分類される。

| 種類 | 内容 | 脳領域 |
|------|------|-------|
| **感覚記憶** | 1秒以下、感覚刺激の残像 | 各感覚野 |
| **短期記憶／ワーキングメモリ** | 数十秒、7±2項目 | 前頭前野、頭頂葉 |
| **長期記憶（陳述／宣言的）** | 意識的に想起可能 | |
| ├ エピソード記憶 | 個人的体験 | 海馬、内側側頭葉 |
| └ 意味記憶 | 一般知識・概念 | 側頭葉外側 |
| **長期記憶（非陳述／非宣言的）** | 無意識的 | |
| ├ 手続き記憶 | 技能・習慣 | 線条体、小脳 |
| ├ プライミング | 先行刺激の促進効果 | 大脳新皮質 |
| └ 古典的条件付け | 連合学習 | 扁桃体（情動）、小脳（運動） |

### 10. 自律神経系の経路と機能（1分）
**定義**：自律神経系（autonomic nervous system, ANS）とは、**意識的制御を介さずに内臓・血管・腺・平滑筋を支配し、生体の恒常性（呼吸・循環・消化・体温・代謝）を維持する末梢神経系の一部**。**交感神経系（闘争・逃走）**と**副交感神経系（休息・消化）**の拮抗的な2系統から構成される。

| | 交感神経 | 副交感神経 |
|--|---------|----------|
| 起始 | 胸腰髄（T1-L2） | 脳幹（III, VII, IX, X）、仙髄（S2-S4） |
| 神経節 | 脊髄近傍（短い節前線維） | 標的臓器近傍（長い節前線維） |
| 節前神経伝達物質 | アセチルコリン（ニコチン受容体） | アセチルコリン（ニコチン受容体） |
| 節後神経伝達物質 | **ノルアドレナリン**（α・β受容体） | **アセチルコリン**（ムスカリン受容体） |
| 機能 | **闘争・逃走**：心拍↑、瞳孔散大、気管支拡張、消化抑制、汗分泌、立毛 | **休息・消化**：心拍↓、瞳孔縮小、消化促進、排尿排便促進 |
※汗腺の交感神経節後はアセチルコリン作動性（例外）。副腎髄質はアドレナリンを血中分泌。

### 11. 神経伝達物質の概要：機能と代謝（1分）
**定義**：神経伝達物質（neurotransmitter）とは、**ニューロンのシナプス前終末から放出され、シナプス後膜の受容体に結合して情報伝達を媒介する化学物質**。化学構造によりモノアミン類、アミノ酸類、コリン類、ペプチド類、ガス分子などに分類される。

- **モノアミン類**：ドーパミン（DA）、ノルアドレナリン（NA）、セロトニン（5-HT）、ヒスタミン
  - 合成：チロシン→DOPA→DA→NA→アドレナリン／トリプトファン→5-HTP→5-HT
  - 代謝：MAO・COMTで分解、代謝物はDA→**HVA**、NA→**MHPG/VMA**、5-HT→**5-HIAA**として尿・髄液で測定可
- **アミノ酸類**：グルタミン酸（最大の興奮性）、GABA（最大の抑制性）、グリシン
- **アセチルコリン**：コリン＋アセチルCoA→AChE で分解
- **神経ペプチド**：エンドルフィン、サブスタンスP、CRH、オキシトシン
- 受容体結合→第二メッセンジャー（cAMP、Ca²⁺）→細胞応答→再取り込み（SERT、DAT、NET）または酵素分解

### 12. モノアミン仮説（30秒）
**定義**：モノアミン仮説（monoamine hypothesis）とは、**うつ病・統合失調症などの主要な精神疾患は、シナプス間隙におけるモノアミン系神経伝達物質（ドーパミン・ノルアドレナリン・セロトニン）の量的異常によって生じる**とする仮説。1960年代に提唱され、抗うつ薬・抗精神病薬の作用機序の基盤となった。
- うつ病：5-HT、NA、DA の不足
- 統合失調症：中脳辺縁系のDA過剰（陽性症状）、中脳皮質系のDA不足（陰性症状）
- 不安症：5-HT 不足、NA 過剰、GABA 不足
- 多くの向精神薬の作用機序を説明できるが、効果発現の遅延（数週間）など説明しきれない部分もあり、現在は**受容体感受性変化説・神経新生説・BDNF説**などへ拡張

### 13. ドーパミン（1分）
- **(1)解剖**：黒質緻密部→線条体（黒質線条体路、運動）／腹側被蓋野→側坐核（中脳辺縁系路、報酬・動機）／腹側被蓋野→前頭前野（中脳皮質路、認知）／視床下部→下垂体（漏斗下垂体路、プロラクチン抑制）
- **(2)機能**：運動制御、報酬・快感、動機付け、注意、ワーキングメモリ
