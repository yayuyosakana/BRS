# 予習ノート：「認知行動療法と精神疾患その２（うつ・双極症）」Ch12

## キーワード
うつ病（大うつ病性障害）、双極症Ⅰ型・Ⅱ型、気分循環症、持続性抑うつ症、SWAG、仮面うつ病、自殺リスク、リチウム、ECT、認知行動療法

---

## 1. 気分障害の分類（DSM-5-TR）

```
気分障害（旧分類）
├── 双極症及び関連症群
│   ├── 双極症Ⅰ型（躁＋うつ）
│   ├── 双極症Ⅱ型（軽躁＋うつ）
│   └── 気分循環症（軽躁様＋気分変調）
└── 抑うつ症群
    ├── うつ病（大うつ病性障害）
    ├── 持続性抑うつ症（気分変調症）
    └── 月経前不快気分障害（PMDD）
```

---

## 2. 各疾患のエピソード持続基準

| 疾患 | エピソードの持続期間 |
|------|------------------|
| うつ病 | うつエピソードが**2週間以上** |
| 双極症Ⅰ型 | **躁エピソード ≥1週間**（入院を要する/精神病症状があれば期間不問）＋うつ |
| 双極症Ⅱ型 | **軽躁エピソード ≥4日間**＋うつエピソード（入院不要、精神病症状なし） |
| 持続性抑うつ症（気分変調症） | 抑うつ気分が**2年以上**（小児は1年） |
| 気分循環症 | 軽躁様・気分変調が**2年以上**（小児は1年） |

---

## 3. うつ病の症状（SWAG）

| 文字 | 症状 |
|------|------|
| **S** | **Suicidality（自殺傾向）** |
| **W** | **Weight loss（体重変化）**（または体重増加） |
| **A** | **Anhedonia（無快楽症）**：以前楽しめたことに興味がない |
| **G** | **Guilt（罪悪感）**：過剰または不適切な罪悪感 |

### SWAG以外のうつ症状
- 抑うつ気分（ほぼ毎日、ほとんど一日中）
- 不眠または過眠
- 精神運動抑制または焦燥
- 疲労・気力低下
- 思考力・集中力の低下
- 死への反復思考

---

## 4. 特殊な形態

### 仮面うつ病（Masked Depression）
- うつ症状ではなく、**身体症状（頭痛・腹痛・疲労感など）**を主訴に受診
- 半数程度のうつ病患者がこの形態
- プライマリケアで見逃されやすい

### 季節性うつ病（Seasonal Affective Disorder: SAD）
- 秋〜冬に発症、春に軽快（光量の減少が原因）
- 非定型症状：過眠・過食・体重増加・鉛様の倦怠感
- 治療：**光療法（高輝度光曝露）**が有効

### 月経前不快気分障害（PMDD）
- 月経前の1週間に激しい情動変化・不快症状が反復
- 月経開始後数日で軽快

## 5. 自殺リスク

### 自殺の危険因子（記憶法：SAD PERSONS）
| 因子 | 内容 |
|------|------|
| 重大な自殺企図歴 | 過去の試みが最大のリスク因子 |
| 45歳以上 | 中高年男性のリスクが高い |
| アルコール依存 | 衝動性・判断力低下 |
| レイプ・暴行歴 | PTSDとの関連 |
| **男性** | 男性の自殺既遂率は女性の3〜4倍（女性は企図が多い） |
| 精神病性症状 | うつ病・統合失調症の精神病性特徴 |
| 独居・社会的孤立 | |
| 具体的な計画 | 手段・日時・場所が決まっている |

## 6. 治療

### 薬物療法

| 疾患 | 第一選択薬 |
|------|----------|
| うつ病 | **SSRI**（フルオキセチン・セルトラリンなど） |
| 双極症Ⅰ型（躁状態） | **リチウム（Lithium）**、バルプロ酸、非定型抗精神病薬 |
| 双極症（維持療法） | リチウム（再発予防に最も強いエビデンス） |

#### リチウムの注意点
- **治療域が狭い**（0.6〜1.2 mEq/L）→ 定期的血中濃度モニタリング必須
- 腎機能・甲状腺機能の定期検査（甲状腺機能低下症を引き起こすことがある）
- 中毒症状：振戦・嘔吐・下痢・意識障害・けいれん

### 心理療法
- **認知行動療法（CBT）**：「否定的な自動思考」を同定・修正
- **対人関係療法（IPT）**：対人関係の問題に焦点を当てる
- どちらもうつ病に対して薬物療法と同等の効果あり（中等度うつに有効）

### ECT（電気けいれん療法）
適応：
- 重篤なうつ病で抗うつ薬が無効・副作用で使えない
- 高齢者
- 自殺の差し迫った危険
- 精神病性うつ病

副作用：一時的な短期記憶障害（通常は回復する）

## 8. 病因

### 生物学的要因
- **神経伝達物質**：セロトニン↓、ノルアドレナリン↓、ドーパミン↓
- **遺伝**：双極症で特に遺伝性が高い（一卵性双生児で約65〜80%一致）
- **HPA軸異常**：コルチゾール過剰→DST陽性

### 心理社会的要因（うつ病）
- 小児期の親との死別・分離
- 成人期の配偶者・子どもの喪失
- 低い自尊感情
- 学習性無力感

## 学習項目への回答

### 1. 気分変調、うつ、軽躁、躁について（30秒）
**定義**：いずれも気分（mood、持続的な情緒状態）の異常を表す精神医学用語である。重症度・方向性・持続期間で区別される。
- **気分変調（dysthymia）**：軽度〜中等度の慢性的な抑うつ気分（DSM-5では「持続性抑うつ症」に統合）
- **うつ（depressive episode）**：抑うつ気分・興味喪失＋身体・認知症状を伴う**2週間以上**の状態
- **軽躁（hypomania）**：気分高揚・活動性亢進が**4日以上**、社会機能障害は軽度、入院不要、精神病症状なし
- **躁（mania）**：気分高揚・易刺激性が**1週間以上**（または入院要する重症度）、著明な社会機能障害、精神病症状を伴うことあり

### 2. 「気分障害」とDSM-5の「双極症及び関連症群」と「抑うつ症群」について（30秒）
DSM-IVでは「気分障害」として一括していたが、**DSM-5では生物学的・遺伝的・経過上の違いを反映して2つに分離**：
- **双極症及び関連症群**：双極症Ⅰ型、Ⅱ型、気分循環症、物質誘発性、他の医学的疾患による
- **抑うつ症群**：うつ病（大うつ病性障害）、持続性抑うつ症（気分変調症）、月経前不快気分障害（PMDD）、重篤気分調節症（DMDD：小児）、物質誘発性、他の医学的疾患による

### 3. 各疾患のエピソード持続期間（1分）
| 疾患 | 必要なエピソードと期間 |
|------|------------------|
| **うつ病** | うつエピソード**≥2週間** |
| **双極症Ⅰ型** | **躁エピソード≥1週間**（入院/精神病症状あれば期間不問）。うつエピソードは診断要件ではない |
| **双極症Ⅱ型** | **軽躁エピソード≥4日間**＋**うつエピソード≥2週間**（少なくとも各1回）、躁エピソードがない |
| **持続性抑うつ症（気分変調症）** | 抑うつ気分が**≥2年間**（小児・青年は1年）、症状のない期間が連続2ヶ月未満 |
| **気分循環症** | 軽躁様症状と気分変調が**≥2年間**（小児・青年は1年）、軽躁・うつ・混合エピソードの基準は満たさない |

### 4. 「抑うつ症群」と「双極症及び関連症群」の生涯有病率について（30秒)
- **うつ病（MDD）**：生涯有病率約**6〜17%**（米国18%、女性は男性の約2倍）
- **持続性抑うつ症**：約**3〜6%**
- **PMDD**：月経のある女性の約**2〜5%**
- **双極症Ⅰ型**：約**0.6〜1.0%**（男女差なし）
- **双極症Ⅱ型**：約**0.4〜1.0%**
- **気分循環症**：約**0.4〜1.0%**

### 5. うつ病の症状SWAGについて（30秒）
**定義**：SWAGとは、うつ病（大うつ病性障害）の中核的な4症状を覚えるための英語の頭文字記憶法である（BRS Behavioral Scienceで採用）。
- **S**uicidality（自殺念慮・自殺企図）
- **W**eight loss/gain or appetite change（体重変化・食欲変化）
- **A**nhedonia（無快楽症：以前楽しめた活動への興味喪失）
- **G**uilt（罪悪感、無価値感）
※DSM-5の9症状中、診断には抑うつ気分または無快楽症のいずれかを含む**5つ以上**が2週間以上必要。

### 6. うつ病のSWAG以外の症状について（30秒）
- **抑うつ気分**：ほぼ毎日、ほとんど一日中
- **不眠または過眠**（中途・早朝覚醒が典型）
- **精神運動性の焦燥または制止**（他覚的）
- **疲労感・気力減退**
- **思考力・集中力の減退、決断困難**
- **死についての反復思考**（自殺念慮・自殺企図・具体的計画）
- 身体症状：頭痛、消化器症状、性欲低下
- 認知症状：絶望感、無力感、自責感

### 7. 仮面うつ病について（30秒)
**定義**：仮面うつ病（masked depression）とは、典型的な抑うつ気分が前景に出ず、身体症状（頭痛、肩こり、腰痛、めまい、動悸、消化器症状、疲労）を主訴として一般診療科を受診するうつ病の臨床型である。中高年男性やアジア人に多いとされる。プライマリケアで見逃されやすく、複数の科を渡り歩く（ドクターショッピング）。**患者は精神科受診を回避**することが多い。治療はSSRIやSNRIに、身体症状への対症療法と心理教育を組み合わせる。

### 8. 季節性うつ病について（30秒）
**定義**：季節性うつ病（seasonal affective disorder, SAD）とは、反復性のうつエピソードが特定の季節（典型的には秋〜冬）に発症し、季節とともに寛解（春〜夏）するうつ病の特定型である（DSM-5では「季節型を伴う」と特定する）。少なくとも**2年連続**で、季節パターンが他の発症パターンを上回る。**非定型症状**（過眠、過食、特に炭水化物渇望、体重増加、鉛様の倦怠感）が多い。**高緯度地域**で多い。原因は、日照時間の短縮によるセロトニン低下と、メラトニン分泌異常である。治療は、**高照度光療法（10,000ルクスを朝30〜60分）**、SSRI、CBTを用いる。

### 9. 月経前不快気分障害について（30秒)
**定義**：月経前不快気分障害（premenstrual dysphoric disorder, PMDD）とは、月経周期の黄体期後期（月経開始前1週間）に重度の情動症状・身体症状が反復出現し、月経開始から数日以内に軽快する、抑うつ症群に分類される精神疾患である（DSM-5）。少なくとも12ヶ月の大半の周期で出現し、社会機能を著明に障害する。

症状：気分の不安定、易刺激性・怒り、抑うつ、不安、興味低下、集中困難、過眠／不眠、過食、身体症状（乳房腫脹、頭痛、関節痛）。治療：**SSRI（黄体期のみまたは持続）**、低用量経口避妊薬、生活習慣（運動、カフェイン制限）、認知行動療法。**PMS（月経前症候群）より重症で診断基準あり**。

### 10. 自殺リスクの5大因子について（30秒)
1. **過去の自殺企図歴**（最強の予測因子）
2. **男性**（既遂率は女性の3〜4倍。女性は企図が3倍多いが致死的方法を選ばない）
3. **精神疾患**（特にうつ病、双極症、統合失調症、物質使用症、PTSD、境界性PD）
4. **物質乱用**（特にアルコール）
5. **社会的孤立／喪失体験**（離婚、死別、失業、社会的支援の欠如）

### 11. 自殺リスクの5大因子以外の因子について（30秒)
- **45歳以上の男性**、**高齢男性**
- **白人・先住民族**
- **未婚・離別・死別**
- **失業・経済的困窮**
- **慢性身体疾患**（特に疼痛、悪性腫瘍、HIV、神経疾患）
- **家族の自殺歴**
- **具体的な計画・手段（特に銃器）の入手可能性**
- **絶望感**（うつ症状より自殺予測力が高い）
- **衝動性・攻撃性**（5-HIAA低値）
- **最近の退院**（特に精神科退院後）
- **小児期トラウマ・性的虐待**
- **LGBTQ（特に思春期）**
- **季節**（春に増加）
- **時刻**（早朝、休日明け）
- **抗うつ薬開始初期**（青年、若年成人）
- **メディアの自殺報道後**（**ウェルテル効果**）

### 12. 双極症の症状について（30秒)
**定義**：双極症（bipolar disorder）とは、躁または軽躁のエピソードと、しばしばうつエピソードが反復する気分障害である。Ⅰ型（躁とうつ）、Ⅱ型（軽躁とうつ）、気分循環症などを含む。

**躁／軽躁エピソード（DIG FAST）**：
- **D**istractibility（注意散漫）
- **I**rresponsibility（無責任な行動：性的逸脱、無謀な投資、運転）
- **G**randiosity（誇大感、自尊心肥大）
- **F**light of ideas（観念奔逸）
- **A**ctivity increased（活動性亢進、目標指向性）
- **S**leep need decreased（睡眠欲求の減少：3時間でも疲れない）
- **T**alkativeness（多弁、急かされた発話）
- **うつエピソード**（ほぼうつ病と同症状）
- 双極症は**うつ期が長く、躁／軽躁期は短い**
- **混合状態**：躁とうつの症状が同時または急速交代→自殺リスク最大

### 13. 精神病性の特徴を伴う双極症およびうつ病について（30秒)
**定義**：精神病性の特徴を伴う気分症（mood disorder with psychotic features）とは、気分エピソード（うつ・躁）の最中に、幻覚または妄想などの精神病症状を伴う重症型である。DSM-5の特定用語である。
- **気分一致性**（mood-congruent）：うつでは罪業妄想・貧困妄想・心気妄想・虚無妄想、躁では誇大妄想・宗教妄想
- **気分不一致性**（mood-incongruent）：気分と無関係な妄想（被害妄想、思考吹入など）→統合失調症との鑑別が重要
- 治療：**抗うつ薬／気分安定薬＋抗精神病薬**併用、**ECTが特に有効**

### 14. 「抑うつ症群」と「双極症及び関連症群」の生物学的病因について（30秒)
- **モノアミン仮説**：うつではセロトニン・ノルアドレナリン・ドーパミン↓、躁では↑
- **HPA軸異常**：コルチゾール↑、**DST（デキサメタゾン抑制試験）非抑制**（うつの約50%）
- **甲状腺機能低下**は抑うつ症状を呈する
- **神経新生・BDNF低下仮説**：海馬の萎縮、抗うつ薬は神経新生を促進
- **概日リズム異常**：睡眠相前進（早朝覚醒）、季節性
- **神経画像**：扁桃体過活動、前頭前野・海馬の活動低下、前頭辺縁系回路の機能異常
- **遺伝因**：**双極症は遺伝率約60〜80%**（うつ病は30〜40%より高い）
- **炎症仮説**：サイトカイン（IL-6, TNF-α）の関与

### 15. 双極症の遺伝について（30秒)
**全精神疾患中で最も遺伝率が高い**疾患の一つ。
- 一卵性双生児の一致率：**60〜80%**
- 二卵性双生児：約20%
- 第一度近親者の有病率：約10%（一般人口の約10倍）
- 統合失調症との遺伝的重複（共通リスク遺伝子）
- 候補遺伝子：CACNA1C（カルシウムチャネル）、ANK3、BDNFなど
- 単一遺伝子病ではなく、**多遺伝子＋環境要因**の複合

### 16. うつ状態の鑑別診断について（30秒)
**定義**：うつ状態の鑑別診断とは、抑うつ気分や興味喪失などのうつ症状を呈する患者に対し、原因疾患を機能性精神疾患・身体疾患・物質誘発性などから区別して特定する診断的な思考過程である。

- **双極症のうつ期**（治療法が異なる：抗うつ薬単独で躁転リスク）
- **持続性抑うつ症**
- **適応反応症（抑うつ気分型）**
- **正常な悲嘆反応**
- **物質誘発性うつ**：アルコール、ベンゾジアゼピン、コカイン離脱、降圧薬（β遮断薬）、ステロイド、インターフェロン、避妊薬
- **身体疾患**：甲状腺機能低下、副腎不全（アジソン）、クッシング症候群、貧血、ビタミン欠乏（B12、葉酸、D）、悪性腫瘍（特に膵がん）、神経変性疾患（パーキンソン、認知症）、脳血管障害、感染症（HIV、神経梅毒）、自己免疫疾患（SLE）
- **認知症**：「**仮性認知症**」（高齢者うつによる認知機能低下、抗うつ薬で可逆）
- **不安症との合併**

### 17. 「抑うつ症群」の心理社会的病因について（30秒)
- **小児期トラウマ**（虐待、ネグレクト、親との早期分離）
- **重要他者の喪失**（死別、離別）、**愛着スタイル**の不安定
- **慢性ストレス**：仕事、家族関係、貧困
- **学習性無力感**（セリグマン）：制御不能な嫌悪体験
- **認知の歪み**（ベック）：**否定的認知の三徴**＝自己・世界・将来への否定的見方、自動思考、推論の誤り
- **対人関係の困難**：孤立、葛藤、役割移行（出産・退職・離婚）
- **社会的支援の欠如**
- **低自尊感情**、**完璧主義**
- **悲観的説明スタイル**（内的・安定的・全般的帰属）

### 18. うつ病の薬物療法について（30秒)
- **第一選択：SSRI**（セルトラリン、エスシタロプラム、フルオキセチン、パロキセチン、フルボキサミン）：効果発現2〜4週、性機能障害・消化器症状・離脱症状
- **SNRI**（ベンラファキシン、デュロキセチン、ミルナシプラン）：高血圧、慢性疼痛にも有効
- **NaSSA**（ミルタザピン）：鎮静・食欲増進、不眠・食欲不振うつに
- **TCA**（イミプラミン、アミトリプチリン）：抗コリン副作用・心毒性、過量で致死的
- **MAOI**：チラミン制限食、相互作用→現代では限定的
- **新規薬**：ボルチオキセチン、ブプロピオン
- **増強療法**：抗精神病薬（アリピプラゾール）、リチウム、甲状腺ホルモン
- **治療抵抗性**：ケタミン／エスケタミン
- **青年・若年成人で自殺念慮増加リスク（ブラックボックス警告）**：開始初期に注意

### 19. 双極症の薬物療法について（30秒)
- **躁エピソード急性期**：リチウム、バルプロ酸、**非定型抗精神病薬**（オランザピン、リスペリドン、クエチアピン、アリピプラゾール）、ベンゾジアゼピン（鎮静）
- **双極性うつ**：**クエチアピン**、ルラシドン、オランザピン＋フルオキセチン、ラモトリギン。**抗うつ薬単独は禁忌**（躁転・急速交代化リスク）
- **維持療法・再発予防**：**リチウム（最強エビデンス、自殺予防効果あり）**、ラモトリギン（うつ予防に強い）、バルプロ酸、抗精神病薬
- **リチウム**：治療域0.6〜1.2 mEq/L、定期血中濃度・腎・甲状腺モニタリング、催奇形性（エブスタイン奇形）
- **バルプロ酸**：肝障害、血小板減少、**催奇形性（神経管閉鎖障害）→妊娠可能女性回避**
- **カルバマゼピン**：再生不良性貧血、Stevens-Johnson症候群（HLA-B*1502陽性アジア人）

### 20. うつ病の認知行動療法について（30秒)
**定義**：うつ病に対する認知行動療法（cognitive behavioral therapy for depression）とは、Beckが体系化した、うつ症状の維持や悪化に関与する否定的な認知（自動思考・スキーマ）と回避的な行動を、構造化されたセッションを通じて修正する短期心理療法である。
- **認知の三徴（cognitive triad）**：①自己（無価値）、②世界（敵対的）、③将来（絶望）
- 推論の誤り（**全か無か思考、過度の一般化、心のフィルター、結論の飛躍、破局視、個人化、すべき思考**など）の同定
- **構造化セッション（12〜20回、週1回）**
- 技法：自動思考記録表、認知再構成、行動活性化（楽しい活動の計画的増加）、行動実験、問題解決訓練、リラプス予防
- **薬物療法と同等の効果**（中等度うつ病、再発予防効果は薬物より高い）

### 21. うつ病の対人関係療法について（30秒)
**定義**：対人関係療法（interpersonal psychotherapy, IPT）とは、KlermanとWeissmanが開発した、現在の対人関係の問題に焦点を当ててうつ症状を改善する、時間制限的（12〜16セッション）な短期心理療法である。うつ病は対人関係の困難と関連するという仮定に基づく。
- **12〜16セッション、短期構造化**
- 4つの問題領域に焦点：
  1. **悲嘆**（重要他者の死別）
  2. **役割移行**（結婚、出産、退職、離別、転職）
  3. **対人関係の役割葛藤**（パートナー・家族・職場）
  4. **対人関係の欠如**（孤立、社交スキル不足）
- 過去の対人関係パターンの理解、コミュニケーション分析、ロールプレイ
- **CBTと同等の効果**、産後うつにも有効

### 22. うつ病の精神療法について（30秒)
**定義**：うつ病の精神療法（psychotherapy for depression）とは、薬物以外の対話や心理的介入を通じて、うつ症状の改善と再発予防を目指す治療法の総称である。エビデンスに基づき複数の技法が用いられる。

- **支持的精神療法**：すべてのケアの基本、傾聴・共感・心理教育・希望の付与
- **CBT**：エビデンス最大
- **対人関係療法（IPT）**：CBTと同等
- **マインドフルネス認知療法（MBCT）**：**再発予防に有効**
- **行動活性化療法（BA）**：CBTから発展、楽しい活動・達成感のある活動の増加
- **問題解決療法**
- **力動的精神療法**：エビデンスは限定的だが慢性うつ・対人関係の問題に
- **集団療法**：心理教育＋ピアサポート
- **家族療法**：家族の高EE（high expressed emotion）の修正
- **運動療法**（軽〜中等症で薬物・心理療法と同等の効果）

### 23. 電気けいれん療法について（30秒)
**定義**：電気けいれん療法（electroconvulsive therapy, ECT）とは、全身麻酔と筋弛緩のもとで頭部に短時間の電気刺激を与え、人為的に全般性てんかん発作を誘発することによって、重症の気分症や精神病性症状を改善させる治療法である。最も即効性の高い抗うつ治療である。
- **適応**：
  - 治療抵抗性のうつ病
  - **重症のうつ病**（強度の自殺念慮・拒食・脱水で緊急性）
  - **精神病性うつ**
  - **緊張病**
  - 妊娠中・薬物使用できないうつ
  - 高齢者（薬物副作用リスクが高い場合）
  - 双極症の急性躁状態（治療抵抗性）
- **手技**：全身麻酔＋筋弛緩薬（修正型ECT）、週2〜3回×6〜12回
- **作用機序**：不明（神経新生、神経可塑性、HPA軸正常化、神経伝達物質増加など仮説）
- **効果**：**最も即効性のある抗うつ治療**（数日で効果）、寛解率70〜80%
- **副作用**：短期記憶障害（通常は数週で回復）、混乱（直後）、頭痛、筋肉痛、麻酔関連リスク
- **禁忌**：絶対禁忌はないが、頭蓋内圧亢進・最近の心筋梗塞・重度脳血管障害は相対的禁忌
- 維持ECT（月1〜2回）で再発予防
